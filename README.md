# Plant Disease Recognition System

AI based plant disease detection project using Deep Learning and TensorFlow.

## Features
- Upload plant leaf image
- Detect disease
- Predict plant health condition

## Technologies Used
- Python
- TensorFlow
- Keras
- Streamlit
- Machine Learning

## How to Run

Install requirements:

pip install -r requirements.txt

Run:

streamlit run app.py
